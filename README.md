# Premium Checkers

Интерактивный веб-сервис шашек 8x8 с акцентом на качество игрового опыта: понятный UI, тактические подсказки, история партии, профиль игрока и зачатки продуктовой монетизации. Проект сделан на базе игрового рендера Three.js и модульной сцено-логики самого нашего движка Star Guide.

## Финальный результат

- Ссылка на рабочий проект: https://sc-gens-test.vercel.app  
- Ссылка на GitHub-репозиторий: https://github.com/neolite/checkers

## Что мы сделали

- Реализовали полноценную игровую логику русских шашек 8x8 с обязательными взятиями, длинными цепочками, дамками и правилом поражения при пате/повторе.
- Собрали адаптивный UI для браузера и мобильного, включая drag-and-drop, подсветку доступных ходов и понятный state-маршрут игры.
- Добавили режимы игры:
  - игра с AI;
  - локальный мультиплеер на одном устройстве;
  - быстрый режим с короткой обратной связью после партии.
- Внедрили профиль игрока с сохранением локальных данных, тему, городом, историей результатов и настройками скинов.
- Подготовили визуальную платформу для роста продукта: рейтинг, приглашения в игру, монетизационный слой и расширяемый AI-помощник.

## Для кого это и зачем

- Для любителей шашек, которым нужен быстрый и понятный онлайн-сервис без установки.
- Для начинающих игроков — чтобы получать устойчивую и понятную обратную связь по правилам партии.
- Для более опытных — чтобы играть в коротких сессиях и быстрее анализировать слабые и сильные решения.
- Это не просто “доска”, а прототип сервиса: есть личный контур пользователя, социальная оболочка и заготовки под монетизацию/сообщество.

## Почему это ценно

- Игра превращается в сервисный продукт, а не демо-страницу.
- Пользователь получает value сразу в первом запуске: быстрый вход, ясный игровой процесс, сохранение прогресса.
- Архитектура уже готова к следующим шагам: турниры, друзья, рейтинг по городам, платная косметика и AI-ассистент с аналитикой.

## Стек
